# Changelog

## 2.0.0 - 2026-04-26

- **type**: MAJOR
- **title**: カテゴリグループ・予定ビュー・PC レイアウト刷新

### 追加

- カテゴリをグループでまとめて整理できるようになりました
- グループ管理ページ（`/groups`）でグループの作成・編集・削除・並び替えができます
- グループに絵文字を設定し、カテゴリ編集ダイアログでグループを割り当てられるようになりました
- カテゴリ管理ページとフィルターサイドバーにグループフィルターを追加しました
- タスクページに「一覧」と「予定」のビュー切り替えを追加しました
- 予定ビューで期限切れ・当日タスクを知らせるアラートチップが表示されるようになりました
- スマートフォンのボトムナビに「メニュー」を追加し、設定・ヘルプへ直接移動できるようになりました

### 改善

- PC レイアウトをヘッダー + 2カラム構成に刷新しました
- タスク追加・編集ダイアログのカテゴリ選択がグループ別タブ表示になりました
- フィルターサイドバーのカテゴリ選択 UI をグリッド表示に統一しました
- フィルターパネルの各セクション見出しに説明ツールチップを追加しました

## 1.3.0 - 2026-04-07

- **type**: MINOR
- **title**: UI 全面刷新・統計ページ追加

### 追加

- カレンダーページを廃止し、タスクの集計・傾向を確認できる統計ページを新設しました
- タスク完了時に「元に戻す」ボタン付きトーストが表示されるようになりました
- ヘルプページを追加しました（絞り込み・ショートカット・タスクモーダルの説明）

### 改善

- PC 表示で検索・絞り込みカラムを追加し、3カラムレイアウトに変更しました
- スマートフォン表示をボトムナビゲーションとフィルター用ボトムシートに刷新しました
- タスク一覧をフラットなリスト表示に変更し、情報を見やすくしました
- タスクカードに作成・完了・スキップの日時がすべて表示されるようになりました
- ステータスフィルターとお気に入りフィルターのボタンに件数が表示されるようになりました
- カテゴリ件数が現在のフィルター条件に連動して更新されるようになりました
- 日付フィルター適用時、セクションラベルに絞り込み条件の補足が表示されるようになりました
- カテゴリのカラーパレットを拡張しました
- キーボードショートカットとフォームの操作性を改善しました
- 完了・やらないタスクの打ち消し線表示を廃止し、よりすっきりした表示にしました

## 1.2.0 - 2026-03-27

- **type**: MINOR
- **title**: カテゴリ機能強化とタスク UI の改善

### 追加

- カテゴリに説明文を追加できるようになりました
- カテゴリをアーカイブして一覧から非表示にできるようになりました
- 「やらない」にしたタスクを元に戻せるようになりました

### 改善

- カテゴリフィルターで複数のカテゴリを同時に選択できるようになりました
- PC 表示でフィルターパネルがサイドバーに移動し、常時表示されるようになりました
- タスクカードのレイアウトを上段・下段の2段構成に改善し、情報が見やすくなりました
- タスクの追加・編集フォームのレイアウトを改善しました
- スマートフォンでのタスク追加・編集時、キーボード表示時にフォームが画面上部に移動するようになりました
- サイドバーのナビゲーション表示を改善しました

### 修正

- タスク追加モーダルで日付フィルターが意図せず反映されていた問題を修正しました

## 1.1.0 - 2026-03-22

### 追加

- 優先度の代わりにお気に入り（★）でタスクをマークできるようになりました
- カテゴリの表示順を並び替えられるようになりました

### 改善

- タスク画面で日付・キーワードを使ったフィルタリングができるようになりました
- タスクカードに作成日時が表示され、予定日が「昨日」「3日前」などの形式で表示されるようになりました
- タスクの更新が画面に即座に反映され、完了通知が表示されるようになりました
- タスク一覧がバックグラウンドで自動的に更新されるようになりました
- カレンダーの集計表示を見やすく改善しました
- 読み込み中の表示をスケルトン UI に改善しました

### 修正

- スマートフォンでもタスクカードの編集ボタンが正しく表示されるようになりました
