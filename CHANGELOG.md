# Changelog

## 1.4.0 - 2026-04-25

- **type**: MINOR
- **title**: カテゴリグループ機能の追加

### 変更内容

- カテゴリをグループでまとめて整理できるようになりました
- グループ管理ページ（`/groups`）でグループの作成・編集・削除・並び替えができます
- カテゴリ編集ダイアログでグループを設定できるようになりました
- カテゴリ管理ページにグループフィルターを追加しました
- カテゴリ一覧でグループ名がチップ表示されるようになりました

## 1.3.0 - 2026-04-07

- **type**: MINOR
- **title**: UI 全面刷新・統計ページ追加

### 変更内容

- PC 表示で検索・絞り込みカラムを追加し、3カラムレイアウトに変更しました
- スマートフォン表示でヘッダーを廃止し、ボトムナビゲーションとフィルター用ボトムシートに変更しました
- カレンダーページを廃止し、タスクの集計・傾向を確認できる統計ページを新設しました
- タスク一覧のグループ表示を廃止し、フラットなリスト表示に変更しました
- タスク完了時に「元に戻す」ボタン付きトーストが表示されるようになりました
- タスクカードに作成・完了・スキップの日時がすべて表示されるようになりました
- ステータスフィルターとお気に入りフィルターのボタンに件数が表示されるようになりました
- カテゴリ件数が現在のフィルター条件に連動して更新されるようになりました
- 日付フィルター適用時、セクションラベルに絞り込み条件の補足が表示されるようになりました
- カテゴリのカラーパレットが拡張されました
- キーボードショートカットとフォームの操作性が改善されました
- 完了・やらないタスクのタスク名の打ち消し線表示を廃止しました
- ヘルプページを追加しました（絞り込み・ショートカット・タスクモーダルの説明）

## 1.2.0 - 2026-03-27

- **type**: MINOR
- **title**: カテゴリ機能強化とタスク UI の改善

### 変更内容

- カテゴリに説明文を追加できるようになりました
- カテゴリをアーカイブして一覧から非表示にできるようになりました
- カテゴリフィルターで複数のカテゴリを同時に選択できるようになりました
- 「やらない」にしたタスクを元に戻せるようになりました
- PC 表示でフィルターパネルがサイドバーに移動し、常時表示されるようになりました
- タスクカードのレイアウトが上段・下段の2段構成になり、情報が見やすくなりました
- タスクの追加・編集フォームのレイアウトが改善されました
- スマートフォンでタスクを追加・編集するとき、キーボード表示時にフォームが画面上部に移動するようになりました
- サイドバーのナビゲーション表示が改善されました
- タスク追加モーダルで日付フィルターが意図せず反映されていた問題を修正しました

## 1.1.0 - 2026-03-22

- 優先度の代わりにお気に入り（★）でタスクをマークできるようになりました
- タスク画面で日付・キーワード検索がまとめてフィルタリングできるようになりました
- カテゴリの表示順を並び替えられるようになりました
- タスクカードに作成日時が表示され、予定日が「昨日」「3日前」などの形式で表示されるようになりました
- タスクの更新が画面に即座に反映され、完了通知が表示されるようになりました
- タスク一覧がバックグラウンドで自動的に更新されるようになりました
- カレンダーの集計表示が見やすくなりました
- 読み込み中の表示がスケルトン UI に改善されました
- スマートフォンでもタスクカードの編集ボタンが正しく表示されるようになりました

## 1.0.0 - 2026-03-20

- タスクの作成・編集・完了・削除・「やらない」操作
- ドラッグ＆ドロップでタスクの並び替え
- N キーショートカットでタスク作成モーダルを開く
- 日付別セクション表示（過期・今日・未予定・完了・スキップ）
- カテゴリフィルター
- タスク詳細シート（メモ内 URL の自動リンク化）
- タスク検索
- カレンダー画面
- メール・パスワード変更、パスワードリセット
- アカウント削除
- PWA manifest 設定
